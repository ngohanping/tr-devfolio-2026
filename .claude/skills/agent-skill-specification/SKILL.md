# Agent Skill Specification

Defines the specification for agent skills in the [agentskills.io ecosystem](https://agentskills.io/specification), outlining required and optional frontmatter fields, directory structure, file reference guidelines, and best practices for progressive disclosure.

## When to Use This Skill

Use this skill when:

- Creating a new agent skill from scratch
- Validating a skill directory for spec compliance

## Reference Files

You must always load the following:

- `references/reference-patterns.md` — Best practices for organizing references/, scripts/, assets/
- `references/frontmatter-examples.md` — Valid and invalid frontmatter examples
