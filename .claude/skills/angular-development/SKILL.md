# Angular Development

Build Angular applications from scratch, enhance existing projects, and optimize frontend performance with deep expertise in Angular, TypeScript, RxJS, and modern web development practices.

## Use this skill when

- Building Angular components and pages
- Implementing new features in an Angular application
- Fixing frontend performance, accessibility, or state issues
- Designing client-side data fetching and reactive flows

## Do not use this skill when

- You only need backend API architecture
- You are building native apps outside the web stack
- You need pure visual design without implementation guidance

## Instructions

- Clarify goals, constraints, and required inputs.
- Apply relevant best practices and validate outcomes.
- Provide actionable steps and verification.

## Response Approach

1. **Analyze requirements** for Angular patterns aligned with company coding standards
2. **Ensure compliance** with feature structure (vertical slice), naming conventions, and DI patterns
3. **Provide production-ready code** with proper TypeScript strict typing
4. **Use RxJS observables** with proper error handling using `catchError`
5. **Implement components** with OnPush change detection and `trackBy` for performance
6. **Follow security practices** with Angular sanitization and route guards
7. **Write Jest tests** with mocked dependencies and direct, explicit assertions
8. **Include accessibility** considerations and ARIA patterns
9. **Always reference** coding-standard-knowledge and tailwind-design-system for consistency
